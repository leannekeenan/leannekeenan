<h1 align="center">Boking</h1>
<h3 align="center">A practice project to learn the basics of react</h3>


<p>The intent of this project was to practice building a MERN stack program using Vanilla HTML. CSS & JavaSCript within a React framework using mongo DB for data storage and express.js and mongoose for backend communication</p>
