<h2>
How It's Made:
</h2>

<p>
The website was built using semantic HTML and flexboxes.  The navigation was kept simple and relies exclusively on CSS and HTML, adjusting size using media queries.  The content of the page is meant to illustrate the assorted beverages a customer can order, the origin story of the coffee shop, and a location to help new customers find their new favorite coffee shop in SF.
</p>

<h2>
Lessons Learned:
</h2>

<p>
This project gave a perfect opportunity to play around with grids, allowing the beverage items to align with each other on small viewports and a three-row grid for larger screen sizes.  Fun little CSS tricks were also used to help accentuate the beverage items by allowing the items to highlight when hovered over to illustrate to the user where the mouse is currently hovering, and what item they are currently viewing.
</p>
