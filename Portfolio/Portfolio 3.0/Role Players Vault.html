About the Wireframe

The wireframe is organized by page left to right, and content top to bottom. The main home page illustrates the logo for the site in the upper left cornet and the nav bar to the far right of it provicing maximum space in the header. There will be a second sub header that provides navigation to the social media the site host will provide to allow users to contact the site creators with questions, comments, and concerns. The promary focus of the page will be the name of the site followed by the action buttons for creating or logging into an account.

When hovered over, the nav buttons will change color to green to show they are active, and a drop down menu will appear upon a onclick function. Each tab will represent a different part of the site and each drop down menu will show what is offered.

The login in tab will hold the login in and create account option just as the main page does to allow the user more than one option to create or access their account. Once the tab or the button is clicked, a window will show unscreen with a request to enter your credentials. The window allows the user the option to toggle back and forth from the login to sign in windows, so if a user selects the incorrect tab, they can easily revert back to where they intended to be directed without having to close the window and begin again

Once the user has created or accessed their account, the will be directed back to the main page, but will be able to see the username they created in teh upper left hand corner to show that tehy have in fact signed in.

The builders tab will hold a host of tools including a character builder, monster builder, and item builder - all of which can convert data to pdf files which can be saved to the local storage of the users computer. The character sheet and monster sheets will bare similarity to the orignal D&D character sheet. They will automatically update the skills of the character when a new value is entered in one of the six primary attributes of the character to help reduce the amount of math needed by the user, allowing them to immerse in their fantasy world rather than crunch numbers.

The sources tab will be a link to resource materials that can be handy while defending yourself in an epic dungeon battle. Have access to spell and item properties and damage imapct t help you make decisions in combat or in an investigation.
