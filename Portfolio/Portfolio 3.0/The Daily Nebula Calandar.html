The Daily Nebula Calendar (Powered by NASA)

With a user-friendly interface and thoughtfully curated content, "The Daily Nebula Calendar" is suitable for space enthusiasts of all ages. Whether you're a seasoned astronomer or just someone who appreciates the beauty of the cosmos, this app will ignite your curiosity and inspire a sense of wonder.

About the Code

This code creates a web page that displays daily photos or videos of the universe, powered by the NASA API. Users can visit the page to explore captivating images or videos of space, accompanied by descriptive information. It provides an immersive and educational experience about the wonders of our universe.

The HTML structure defines the layout of the page. It includes meta tags for character encoding, viewport settings, and page description. The title of the page is set as "The Daily Nebula." The code also references an external CSS file for styling purposes. The HTML elements consist of a heading, paragraph, and a section to display the daily content. The section includes an image, iframe (for videos), date, title, and description.

The CSS styling defines the visual appearance of the page. It sets the text alignment, padding, dimensions, box shadow, and border radius for different elements. The styling ensures a centered layout and enhances the visual presentation of the content.

The JavaScript code fetches data from the NASA API by making a fetch request to the specified URL. Upon receiving the data, it updates the HTML elements dynamically. The date is set to the current date using JavaScript's Date object. If the media type is an image, the fetched HD image URL is assigned to the src attribute of the image element. If the media type is a video, the URL is assigned to the src attribute of the iframe element. The title and description are also updated with the corresponding data from the API response.
