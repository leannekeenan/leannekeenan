Lumen Entertainment Presents: War! The Card Game

Introducing "WAR! - The Card Game" - a captivating and engaging web application that brings the excitement of the classic card game right to your fingertips! With its intuitive design and seamless gameplay, this app is perfect for card game enthusiasts looking for some thrilling entertainment.

About the Code

This code demonstrates a basic implementation of the "War!" card game, allowing users to play against the computer by drawing and comparing cards from a shuffled deck.

The provided code represents a simple card game called "War!" implemented using HTML, CSS, and JavaScript. The HTML structure consists of various sections and elements that define the game layout. The CSS styles define the visual appearance of the elements, including background colors, text alignment, and dimensions. The JavaScript code handles the game logic and interaction with the external API.

When the page loads, the user can click the "?" icon to get instructions on how to play the game. The JavaScript code fetches a deck of cards from the "deckofcardsapi.com" API by making an HTTP request and retrieves a unique deck ID. Upon clicking the "Draw Cards" button, another request is sent to the API, specifying the deck ID and the number of cards to draw. The API response provides information about the drawn cards, including their values and images.

The JavaScript code then compares the values of the two drawn cards to determine the winner of the round. If the value of the first card is greater than the second card, the message "Player 1 WINS!" is displayed. Conversely, if the value of the first card is less than the second card, the message "Computer WINS!" is shown. In the event of a tie, the message "WAR!" is displayed.
