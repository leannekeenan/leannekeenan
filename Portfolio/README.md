<h2>
How It's Made:
</h2>

<p>
This page was made using semantic HTML5 tags to ensure proper functionality, readability, and to assist in the 
accessability of the content for screen reader users.  The CSS of the page uses flexboxes to add responsive behavior to 
the page elements during viewport changes. Media Queries have been included to appropriately sizr the viewport based on 
the device the site is viewed onThe page also includes the use of JavaScript in the nav bar element as well as the email 
form allowing visitors to send me messages requesting assistance or a project collaboration partner.  The site also 
showcases a few of my most recent projects to illustrate my abilities.
</p>

<h2>
Lessons Learned:
</h2>

<p>
This project really made me sit and assess myself as a developer and as a person. Building portfolio taught me how to
