<h2>
How It's Made:
</h2>

<p>
Users are asked to enter a question of their choice into the input setcion at the top of the page.
Using an event listener, a 'click' event is waited for on the 'ask'button
Once the ask buttonis clicked, the question is answered by a function that cycles through an array of retorts that
will serve as the answers from the magic eight ball.  The user can reclick the button infinite times to get the 
answer they want much like a real magic eight ball.
</p>

<h2>
Lessons Learned:
</h2>

<p>
This project illustrates the use if if/else statemetns to control the flow of a program.
The if/else statements allow users to cycle through multiple options and choosing one at random based on user input
</p>
