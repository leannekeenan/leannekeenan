<h2>
How It's Made:
</h2>

<p>
Using a fetch request, the content from a NASA API is pulled and entered into the specified HTML tags, including a tag for the title of the image, and the image itself.  
</p>

<h2>
Lessons Learned:
</h2>

<p>
This project shows how to pull a basic API fetch request with a source link to the NASA API for the images, image title, and description of the image.  The use of iframe was also integrated to ensure that if any image produced happens to be video, the content will still render appropriately.  A 'toDateString() function was used for the date element to ensure the date is always correct.
</p>
